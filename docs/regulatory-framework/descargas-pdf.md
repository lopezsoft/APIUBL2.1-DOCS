## 📥 Instrucciones de Descarga

### Pasos Generales

1. **Identificar el documento** que necesitas (FE, Nómina, RADIAN, etc.)
2. **Seleccionar la versión** vigente (todas las mostradas aquí son 2024-2025)
3. **Descargar** desde `/docs-dian/` o desde DIAN oficial
4. **Extraer** si es ZIP (documentos de ejemplo)
5. **Validar** versión y Resolución en encabezado
6. **Usar** como referencia técnica

### Para Desarrolladores

```
VALIDACIÓN:
├─ Comparar versión descargada con esta página
├─ Verificar fecha de Resolución
├─ Revisar firma de PDF (si aplica)
└─ Confirmar checksums (si disponibles)

INTEGRACIÓN:
├─ Usar ejemplos XML para testing
├─ Referencia técnica para especificaciones
├─ Tablas Excel para codificación
└─ Validaciones del Anexo para reglas de negocio
```

### Para Consultores/Auditores

```
CONSULTA:
├─ Anexo Técnico = Referencia normativa
├─ Ejemplos = Casos válidos
├─ Matrices = Catálogos de códigos
└─ Este sitio = Interpretación y guía

AUDITORÍA:
├─ Validar contra especificaciones del Anexo
├─ Usar ejemplos como referencia de validez
├─ Comparar códigos contra matrices
└─ Documentar hallazgos con referencias
```
