# Documento Soporte - Operaciones Efectivo

## Definición

El **Documento Soporte de Operaciones Efectivo** es un documento electrónico que puede servir como comprobante de transacciones con terceros sin costo para el usuario final.

## Base Normativa

| Aspecto | Detalle |
|--------|---------|
| **Resolución** | Resolución 000160 de 2024 DIAN |
| **Versión Vigente** | 1.1 |
| **Aplicabilidad** | Operaciones de efectivo y servicios en línea |

## Características

✅ Comprobante de transacción electrónica
✅ Válido para operaciones de efectivo
✅ Uso en comercio electrónico
✅ Obligatorio para plataformas de pagos

:::info
La documentación completa del Documento Soporte está en desarrollo. Por ahora, consulta los anexos técnicos oficiales de la DIAN.
:::
