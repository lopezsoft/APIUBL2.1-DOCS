# Documentos Equivalentes (DE)

## Definición

Los **Documentos Equivalentes Electrónicos** son documentos emitidos por personas o entidades no obligadas a emitir factura electrónica que tienen el mismo valor probatorio.

## Base Normativa

| Aspecto | Detalle |
|--------|---------|
| **Resolución** | Resolución 000166 de 2024 DIAN |
| **Versión Vigente** | 1.3 |
| **Destinatarios** | Sujetos no obligados a FE |

## Características

✅ Válido para sujetos no obligados a facturación electrónica
✅ Igual valor probatorio que factura electrónica
✅ Formato electrónico obligatorio
✅ Debe estar firmado digitalmente
